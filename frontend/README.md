# Frontend - Portal de Vistoria HaaS

Interface React + TypeScript + Vite

## Setup

1. Instalar dependências:
```bash
npm install
```

2. Criar arquivo `.env`:
```bash
cp .env.example .env
```

3. Preencher `.env` com URL da API

## Desenvolvimento

```bash
npm run dev
```

Aplicação rodará em `http://localhost:5173`

## Build

```bash
npm run build
```

Gera pasta `dist/` pronta para produção

## Páginas

- `/login` - Login
- `/` - Dashboard
